bbscaff 0.0.5
===================

> Scaffold tool for Backbase CXP **5.5.x**

## Install

``` shell
npm install -g bbscaff
```

## Usage

### Generating Widgets
Inside `/yourBundleName/widgets` type:

``` shell
bbscaff generate widget
```

### Generating Containers
Inside `/yourBundleName/containers` type:

``` shell
bbscaff generate container
```
